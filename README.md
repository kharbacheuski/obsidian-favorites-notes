# Favorites Notes
Fast, keyboard-first favorites navigator for <a href="https://obsidian.md">Obsidian</a>.

## Features

- Open your favorite notes from a single modal window.

- Navigate using ↑ / ↓ and open instantly with Enter.

- Always open notes in a new tab for a non-destructive workflow.

- `Ctrl + Enter` opens a note in split view (optional, configurable).

- Visual focus indicators and action hints for power users.

- Drag & drop sorting and custom display titles.

- Designed for speed, keyboard control, and daily workflows.

- Lightweight and fully integrated with Obsidian’s workspace system.
